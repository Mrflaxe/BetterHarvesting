## ✨ Overview

In vanilla Minecraft, clearing a field means breaking each crop and then planting
a seed back, over and over. **BetterHarvesting** does both at once.

**Right‑click a fully grown crop with any hoe** — the plugin collects the drops,
replants the field, and keeps one seed back for the replant, just like you'd do by hand.

## 🚀 Features

| | Feature                                                                     |
|---|-----------------------------------------------------------------------------|
| 🖱️ | **One‑click harvest & replant** — right‑click a grown crop with any hoe     |
| 🌱 | **Auto‑reseed** — keeps a seed aside and replants the field for you         |
| ⛏️ | **Durability consumption** — the hoe takes wear, just like breaking a block |
| ✨ | **Enchantment‑aware** — respects the *Unbreaking* enchantment               |
| 📊 | **Vanilla statistics** — updates your *Mined* and *Used* stat counters      |
| 🏆 | **Advancements** — unlocks *"A Seedy Place"* when you replant               |
| 🔊 | **Familiar sounds** — the usual tilling and planting sounds play as you go  |
| 🪶 | **Zero setup** — install it and you're done                                 |

## 🌿 Supported Crops

🌾 Wheat · 🥕 Carrots · 🥔 Potatoes · 🟥 Beetroots · 🍄 Nether Wart

## 📦 Installation

1. Download the latest `BetterHarvesting.jar` from the [Releases](https://github.com/Mrflaxe/BetterHarvesting/releases) page.
2. Drop it into your server's `plugins/` folder.
3. Restart the server.

## 🧩 Compatibility

- **Server software:** Spigot, Paper, and forks
- **Minecraft versions:** `1.14` and up, including the latest releases (26.1.2)
- **Dependencies:** none

The plugin ships per‑version modules and automatically selects the right one for
your server at startup, so a single jar works across the full version range.
